접근성 이란?

웹 접근성(Web Accessibility) 이란 장애인,고령자 등이 웹 사이트에서 제공하는 정보에 비장애인과 동등하게 접근할 수 있도록 보장하는 것입니다.
우리나라는 한국형 웹 콘텐츠 접근성 지침2.1를 기준으로 하고있습니다.

아래의 목록은 장애인마다 주의점과 보안점을 설명
  • 전맹모니터를 볼수 없음보안대책: 스크린 리더
  • 저시력모니터를 일부 사용보안대책: 화면확대/고대비
  • 색맹색을 구분할 수 없음보안대책: 색상에만 의존하지 않기/고대비
  • 청각장애사운드,오디오 등을 창취할 수 없음보안대첵: 수화,자막등 시각정보 제공
  • 상지장애손을 사용할 수 없음보안대책: 마우스 대체 방법,키보드만 사용
  • 기타움직임이 어려움보안대책: 충분한 시간 제공
  • 언어장애복잡한 용어, 어려운 용어의 이해 불가능보안대책: 쉬운 용어 사용

한국형 웹을 제작을 위한 약속

한국에서는 정부부터가 웹 표준에 부진함이 있지만 웹 서비스를 주력으로 삼는 기업들은 거의 대부분 웹 표준을 따라가려고 노력하고 있다.

아래의 목록은 웹지침관련 단체를 설명합니다.
  • W3C (World Wide Web Consortium)대표적으로 HTML과 CSS의 표준을 정한다.
    마크업에 대한 준수와 태그의 속성과 스타일의 정보를 확인 하고 준수
  • Ecma 인터내셔널 표준JavaScript 표준이 여기서 정해진다. JavsScript를 사용함에 있어 여러 효과와 제어를 접근에 맞춰 확인 하고 준수
  • 우리나라의 웹 접근성현재는 대다수의 웹제작 기업은 한국 웹 연구소의 웹 접근성 지침2.1 를 준수중이다.
    그리고 최근 정부에선 액티브 X, 공인인증서, 플러그인 사용을 없앤다고 발표했다.

cookie

사용자 컴퓨터에 텍스트 파일(최대 4kb)로 저장되는 데이터
쿠키파일은 만료일을 지정할수 있다
하루동안 열리지 않는 팝업창, 쇼핑몰 장바구니 등을 구현
웹서버는 사용자가 웹페이지를 닫으면 접속을 종료하기 때문에 사용자에 대한 정보를 기억해 두기위해서 쿠키 파일을 생성한다

쿠키는 웹 사이트에서 사용자의 기본 설정에 대한 정보를 저장하기 위해 PC에 추가하는 작은 파일. 
쿠키는 사이트에서 사용자의 기본 설정을 기억하도록 허용하거나 사용자가 특정 사이트를 방문할 때마다 로그인할 필요가 없게 함으로써 
검색 환경을 개선한다. 
하지만 일부 쿠키는 사용자가 방문하는 사이트를 추적하므로 개인 정보가 위험해질 수 있습니다.

= 객체.프로퍼티의 형식
쿠키 정보를 가져올때 사용함 => document.cookie
쿠키 정보를 저장하고 싶을때 => document.cookie = '이름1=값1; 이름2=값2';

= 쿠키는 서버에서만 확인됨
크롬 : F12.개발자모드 > Application > Cookies에서 확인됨
IE  : 도구 > 인터넷옵션 > 검색기록 : 설정 > 파일보기
